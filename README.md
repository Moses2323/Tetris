# Tetris

## About

Обыкновенный тетрис. Управление падающими элементами возможно с помощью нажатия
на соответствующие кнопки в интерфейсе, либо с помощью клавиатуры:

a - влево

d - вправо

Space - ускорить движение фигуры

w - поворот по часовой стрелке

p - пауза

Скорость падения фигур можно регулировать с помощью ползунка.

За одну линию дается 10 очков. За две одновременно - 30, за три - 60, за четыре - 100
